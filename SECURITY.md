# Security Policy

## Supported Versions

| Version | Supported          |
|---------| ------------------ |
| 1.3.0   | :white_check_mark: |

## Reporting a Vulnerability

Please report security issues privately via GitHub's
[private vulnerability reporting](https://github.com/status201/orca-dam/security/advisories/new).

Do **not** open a public issue for security-relevant findings.
